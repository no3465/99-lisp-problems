### How to run?
Using `mit-scheme` command
```sh
$ mit-scheme --load "/path/to/scm/file"
```
or
Inside `mit-scheme` REPL
```sh
(load "/path/to/scm/file")
(function-name arg1 arg2 ... argn)
```
